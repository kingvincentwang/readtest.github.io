第1課：CSS 簡介與基本語法

CSS（層疊樣式表）是用來控制網頁外觀的語言。它可以控制顏色、字體、間距、佈局等多種視覺效果。

CSS 語法

selector {
  property: value;
  another-property: another-value;
}

CSS 引入方式

  • 內聯樣式：直接在 HTML 元素中使用 style 屬性
  • 內部樣式表：在 HTML 文件的 <head> 使用 <style>
  • 外部樣式表：使用 <link> 引入 .css 文件

實例

HTML
<div class="box">
  這是一個 CSS 示例
</div>
CSS
.box {
  background-color: #3b82f6;
  color: white;
  padding: 20px;
  border-radius: 8px;
  text-align: center;
}
這是一個 CSS 示例

第2課：選擇器與優先級

CSS 選擇器可用來選取 HTML 元素並套用樣式：

p { color: red; }           /* 標籤選擇器 */
.box { padding: 10px; }     /* 類別選擇器 */
#header { background: blue; } /* ID 選擇器 */

優先級：inline style > ID > class > tag

第3課：顏色與背景

CSS 提供多種方式來設定顏色和背景，掌握這些技巧可以讓網頁更加美觀。

顏色表示方法

/* 顏色名稱 */
color: red;
color: blue;
color: forestgreen;

/* 十六進制 */
color: #ff0000;  /* 紅色 */
color: #00ff00;  /* 綠色 */
color: #0000ff;  /* 藍色 */
color: #333;     /* 深灰色 (簡寫) */

/* RGB */
color: rgb(255, 0, 0);      /* 紅色 */
color: rgb(0, 255, 0);      /* 綠色 */

/* RGBA (帶透明度) */
color: rgba(255, 0, 0, 0.5);  /* 半透明紅色 */
color: rgba(0, 0, 0, 0.8);    /* 80% 不透明黑色 */

/* HSL (色相、飽和度、亮度) */
color: hsl(0, 100%, 50%);     /* 紅色 */
color: hsl(120, 100%, 50%);   /* 綠色 */

背景屬性

/* 背景顏色 */
background-color: #f0f4f8;
background-color: rgba(255, 255, 255, 0.9);

/* 背景圖片 */
background-image: url("background.jpg");
background-image: linear-gradient(to right, #ff7e5f, #feb47b);

/* 背景重複 */
background-repeat: no-repeat;  /* 不重複 */
background-repeat: repeat-x;   /* 水平重複 */
background-repeat: repeat-y;   /* 垂直重複 */

/* 背景位置 */
background-position: center;
background-position: top left;
background-position: 50% 25%;

/* 背景大小 */
background-size: cover;        /* 覆蓋整個容器 */
background-size: contain;      /* 完整顯示圖片 */
background-size: 100% 100%;    /* 拉伸填滿 */

/* 簡寫語法 */
background: #fff url("bg.jpg") no-repeat center/cover;

漸變背景

/* 線性漸變 */
background: linear-gradient(to right, #ff7e5f, #feb47b);
background: linear-gradient(45deg, #667eea, #764ba2);
background: linear-gradient(to bottom, red 0%, yellow 50%, green 100%);

/* 徑向漸變 */
background: radial-gradient(circle, #ff7e5f, #feb47b);
background: radial-gradient(ellipse at center, #667eea 0%, #764ba2 100%);

顏色理論基礎

RGB 色彩模式

紅(Red)、綠(Green)、藍(Blue)三原色混合

HSL 色彩模式

色相(Hue)、飽和度(Saturation)、亮度(Lightness)

色相: 0-360度
飽和度: 0-100%
亮度: 0-100%

實際應用範例

純色背景

background-color: #3b82f6;

漸變背景

線性漸變效果

半透明背景

rgba(255, 193, 7, 0.2)

左：線性漸變 45度，右：圓形徑向漸變

💡 實用小技巧
  • 使用 RGBA 可以創建半透明效果，適合疊加設計
  • HSL 比 RGB 更直觀，適合調整顏色明暗
  • 漸變可以替代圖片，減少檔案大小
  • background-size: cover 確保圖片覆蓋整個容器

第5課：盒模型：邊距、邊框與內邊距

盒模型是 CSS 佈局的基礎概念。每個元素都是一個矩形盒子，由內容、內邊距、邊框和外邊距組成。

盒模型組成

內容區域 (content)
內邊距 (padding)
邊框 (border)
外邊距 (margin)

外邊距 (margin)

/* 四邊相同 */
margin: 20px;

/* 上下、左右 */
margin: 10px 20px;

/* 上、左右、下 */
margin: 10px 20px 15px;

/* 上、右、下、左 (順時針) */
margin: 10px 20px 15px 5px;

/* 單獨設定 */
margin-top: 10px;
margin-right: 20px;
margin-bottom: 15px;
margin-left: 5px;

/* 水平置中 */
margin: 0 auto;

內邊距 (padding)

/* 語法與 margin 相同 */
padding: 15px;
padding: 10px 20px;
padding-top: 10px;

邊框 (border)

/* 簡寫語法 */
border: 2px solid #333;

/* 分別設定 */
border-width: 2px;
border-style: solid; /* solid, dashed, dotted, double */
border-color: #333;

/* 圓角 */
border-radius: 8px;
border-radius: 50%; /* 圓形 */

box-sizing 屬性

/* 改變盒模型計算方式 */
box-sizing: content-box; /* 預設值，寬度不含 padding 和 border */
box-sizing: border-box;  /* 寬度包含 padding 和 border */

/* 建議全域設定 */
* {
  box-sizing: border-box;
}

第6課：尺寸與單位

CSS 支援多種單位，選擇合適的單位對響應式設計和易維護性非常重要。

絕對單位

/* 像素 - 最常用 */
width: 200px;
font-size: 16px;

/* 其他絕對單位 (較少使用) */
width: 2in;   /* 英寸 */
width: 5cm;   /* 公分 */
width: 12pt;  /* 點 (印刷業) */

相對單位

/* em - 相對於父元素字體大小 */
font-size: 1.2em;
margin: 0.5em;

/* rem - 相對於根元素字體大小 */
font-size: 1.5rem;
padding: 2rem;

/* 百分比 - 相對於父元素 */
width: 80%;
height: 50%;

/* 視窗單位 */
width: 100vw;  /* 視窗寬度的 100% */
height: 100vh; /* 視窗高度的 100% */
font-size: 4vw; /* 響應式字體 */

/* 其他單位 */
width: 10ch;   /* 字符寬度 */
height: 1ex;   /* x 字母高度 */

寬度和高度

/* 基本尺寸 */
width: 300px;
height: 200px;

/* 最大最小值 */
max-width: 100%;
min-width: 200px;
max-height: 500px;
min-height: 100px;

/* 特殊值 */
width: auto;     /* 自動 */
width: inherit;  /* 繼承父元素 */
width: initial;  /* 初始值 */
width: fit-content; /* 適合內容 */

第7課：顯示與定位

display 和 position 屬性控制元素的顯示方式和定位行為，是佈局的關鍵屬性。

display 屬性

/* 塊級元素 - 佔滿一行 */
display: block;

/* 行內元素 - 在同一行顯示 */
display: inline;

/* 行內塊級 - 結合兩者特點 */
display: inline-block;

/* 隱藏元素 */
display: none;

/* 彈性盒子 */
display: flex;

/* 網格佈局 */
display: grid;

/* 表格顯示 */
display: table;
display: table-cell;

position 屬性

/* 預設值 - 正常文檔流 */
position: static;

/* 相對定位 - 相對於原位置 */
position: relative;
top: 10px;
left: 20px;

/* 絕對定位 - 相對於最近的定位祖先 */
position: absolute;
top: 0;
right: 0;

第8課：浮動與清除

浮動 (float) 是早期 CSS 佈局的主要方式，雖然現在有更好的方法，但理解浮動仍然重要。

浮動基礎

/* 浮動方向 */
float: left;   /* 向左浮動 */
float: right;  /* 向右浮動 */
float: none;   /* 不浮動 (預設) */

.image {
  float: left;
  margin: 0 20px 20px 0;
}

清除浮動

/* clear 屬性 */
clear: left;   /* 清除左浮動 */
clear: right;  /* 清除右浮動 */
clear: both;   /* 清除所有浮動 */

/* clearfix 技巧 - 解決父元素高度塌陷 */
.clearfix::after {
  content: "";
  display: table;
  clear: both;
}

/* 現代 clearfix */
.clearfix::after {
  content: "";
  display: block;
  clear: both;
}

浮動佈局範例

/* 兩欄佈局 */
.sidebar {
  float: left;
  width: 25%;
}

.main-content {
  float: right;
  width: 70%;
}

/* 三欄佈局 */
.left-sidebar { float: left; width: 20%; }
.right-sidebar { float: right; width: 20%; }
.main { margin: 0 22%; } /* 左右留空間 */

這是一個浮動範例。藍色方塊向左浮動，文字會環繞在它的右邊和下方。這就是浮動的基本效果。

為什麼需要重置？

  • 確保跨瀏覽器的一致性

  height: auto;

🔸 Flexbox 彈性盒子佈局

Flexbox 基礎概念

Flexbox 是一維佈局方法，主要用於在容器中排列項目，提供靈活的空間分配和對齊控制。

/* 啟用 Flexbox */
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 20px;
}

/* 子項目屬性 */
.item {
  flex: 1;  /* 等比例擴展 */
}

.item-grow {
  flex-grow: 2;  /* 擴展比例為 2 */
}

主要屬性詳解

父容器屬性
  • justify-content: 主軸對齊
  • align-items: 交叉軸對齊
  • flex-direction: 主軸方向
  • flex-wrap: 是否換行
  • gap: 項目間距
子項目屬性
  • flex-grow: 擴展比例
  • flex-shrink: 收縮比例
  • flex-basis: 基準大小
  • align-self: 個別對齊
  • order: 排列順序

🔸 Media Queries 媒體查詢

基本語法

Media Queries 讓我們能夠根據設備特性（如螢幕寬度、高度、方向等）套用不同的 CSS 樣式。

/* 基本語法 */
		@media media-type and (media-feature) {
		  /* CSS 規則 */
		}

		/* 常用範例 */
		@media screen and (max-width: 768px) {
		  .container {
			width: 100%;
			padding: 0 15px;
		  }
		}

		@media screen and (min-width: 1200px) {
		  .container {
			max-width: 1140px;
			margin: 0 auto;
		  }
		}

		/* 方向查詢 */
		@media screen and (orientation: landscape) {
		  .sidebar {
			width: 25%;
		  }
		}

		@media screen and (orientation: portrait) {
		  .sidebar {
			width: 100%;
		  }
		}

常用媒體特性

尺寸相關
  • width / height - 確切尺寸
  • max-width / min-width - 最大/最小寬度
  • device-width - 設備寬度
  • aspect-ratio - 長寬比
其他特性
  • orientation - 方向（portrait/landscape）
  • resolution - 解析度
  • hover - 是否支援懸停
  • pointer - 主要輸入機制

函數 (Functions)

// 自定義函數
@function px-to-rem($px, $base-font-size: 16px) {
  @return ($px / $base-font-size) * 1rem;
}

@function str-replace($string, $search, $replace: '') {
  $index: str-index($string, $search);
  
  @if $index {
    @return str-slice($string, 1, $index - 1) + $replace + 
            str-replace(str-slice($string, $index + str-length($search)), $search, $replace);
  }
  
  @return $string;
}

// 使用函數
.title {
  font-size: px-to-rem(24px);  // 輸出: 1.5rem
  margin-bottom: px-to-rem(16px);  // 輸出: 1rem
}

OOCSS 將 CSS 視為物件導向程式設計，強調分離結構和外觀、容器和內容。

原則 1：分離結構和外觀
/* 結構 */
.btn {
  display: inline-block;
  padding: 8px 16px;
  border: none;
  border-radius: 4px;
  cursor: pointer;
}

/* 外觀 */
.btn-primary { background: #007bff; color: white; }
.btn-secondary { background: #6c757d; color: white; }
.btn-success { background: #28a745; color: white; }
原則 2：分離容器和內容
/* 避免依賴位置的樣式 */
/* ❌ 不好的做法 */
.sidebar h3 { color: blue; }

/* ✅ 好的做法 */
.title-blue { color: blue; }

/* 使用 */
<h3 class="title-blue">標題</h3>

元素的狀態變化

Theme 主題

視覺主題相關

.theme-dark

🔸 常見陷阱與解決方案

❌ 全域污染

問題：

/* 過於通用的樣式 */
h3 { color: blue; }
.content p { margin-bottom: 20px; }

解決：

/* 使用特定的類別 */
.article__title { color: blue; }
.article__paragraph { margin-bottom: 20px; }
❌ 過度使用 !important

問題：

.text { color: blue !important; }
.highlight { color: red !important; }

解決：

/* 使用更具體的選擇器 */
.article .text { color: blue; }
.article .text.highlight { color: red; }
❌ 硬編碼數值

問題：

.card { margin: 16px; }
.button { padding: 12px 24px; }
.header { height: 64px; }

解決：

/* 使用 CSS 變數 */
.card { margin: var(--space-md); }
.button { padding: var(--space-sm) var(--space-lg); }
.header { height: var(--header-height); }

邊框相關屬性
